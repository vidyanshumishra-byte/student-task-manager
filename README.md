# 🚀 Student Task Manager v4 (Stable Pro)

A sleek, modern, and lightweight Desktop Application designed to help students track their daily tasks, deadlines, and priorities without any clutter. Built with a beautiful dark theme using Python and CustomTkinter.

---

## ✨ Key Features
* **Modern UI:** Discord/Spotify style dark mode interface with rounded corners and smooth buttons.
* **Smart Prioritization:** Categorize tasks by **High (🔴), Medium (🟡), and Low (🟢)** priority with color-coded indicators.
* **Progress Tracking:** Live progress bar showing completed vs. total tasks.
* **Persistent Storage:** Your tasks are saved automatically to a local `tasks.json` file. You won't lose data when you close the app.
* **Due Dates:** Easily assign and view due dates for upcoming assignments or exams.
